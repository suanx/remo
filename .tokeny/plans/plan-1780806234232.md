# Rust LLM Agent 框架

## Context
用 Rust 构建一个生产级 LLM Agent 框架，支持 OpenAI 兼容 API、ReAct 推理循环、工具调用、流式输出，提供 CLI 和 HTTP Server 两种使用方式。

## 技术选型
| 依赖 | 版本 | 用途 |
|------|------|------|
| tokio | 1 (full) | 异步运行时 |
| reqwest | 0.12 | HTTP 客户端（含 SSE 流式） |
| axum | 0.8 | HTTP 服务端 |
| serde / serde_json | 1 | JSON 序列化 |
| async-trait | 0.1 | 异步 trait 支持 |
| rustyline | 15 | CLI 行编辑 |
| colored | 3 | 终端彩色输出 |
| futures-util | 0.3 | Stream 工具（SSE 流处理） |
| tracing / tracing-subscriber | 0.1 | 日志 |

## 核心接口设计

### 1. LLM 层 (`src/llm/`)
```rust
// src/llm/message.rs
pub enum Role { System, User, Assistant, Tool }

pub struct Message {
    pub role: Role,
    pub content: Option<String>,
    pub tool_calls: Vec<ToolCall>,       // assistant 发起的调用
    pub tool_call_id: Option<String>,    // tool 回复对应的 call id
    pub name: Option<String>,            // tool name
}

pub struct ToolCall {
    pub id: String,
    pub function: FunctionCall,
}

pub struct FunctionCall {
    pub name: String,
    pub arguments: String,  // JSON string
}

pub struct FunctionDef {
    pub name: String,
    pub description: String,
    pub parameters: serde_json::Value,  // JSON Schema
}
```

```rust
// src/llm/client.rs
#[async_trait]
pub trait LlmClient: Send + Sync {
    async fn chat(&self, messages: &[Message], tools: &[FunctionDef]) -> Result<Message>;
    async fn chat_stream(
        &self, messages: &[Message], tools: &[FunctionDef],
        on_token: &dyn Fn(&str) + Send,
    ) -> Result<Message>;
}
```
- `OpenAiClient`：`{ base_url, api_key, model }`，reqwest POST `/chat/completions`
- 流式：`stream=true`，逐行解析 `data: {...}` SSE 事件，累积 delta 拼成完整 Message
- 用 `futures_util::BufReader` + `lines()` 逐行读取 SSE 流

### 2. Agent 层 (`src/agent/`)
```rust
// src/agent/config.rs
pub struct AgentConfig {
    pub model: String,
    pub system_prompt: String,
    pub max_turns: usize,
}
```

```rust
// src/agent/mod.rs
pub struct Agent {
    client: Box<dyn LlmClient>,
    tools: ToolRegistry,
    config: AgentConfig,
}

impl Agent {
    pub async fn run(&self, user_input: &str) -> Result<String>;
    pub async fn run_stream(&self, user_input: &str, on_token: &dyn Fn(&str) + Send) -> Result<String>;
}
```

**ReAct 循环逻辑：**
```
messages = [system_prompt, user_input]
loop {
    response = llm.chat(messages, tool_defs)
    if no tool_calls → return response.content
    messages.push(response)           // assistant message with tool_calls
    for call in response.tool_calls:
        result = registry.execute(call.name, call.arguments)
        messages.push(tool_result_message(call.id, result))
    if turns >= max_turns → return "已达最大推理轮次"
}
```

### 3. 工具层 (`src/tools/`)
```rust
#[async_trait]
pub trait Tool: Send + Sync {
    fn name(&self) -> &str;
    fn description(&self) -> &str;
    fn parameters(&self) -> serde_json::Value;
    async fn execute(&self, args: serde_json::Value) -> Result<String>;
}
```

内置工具：
| 工具 | 说明 | 参数 |
|------|------|------|
| `calculator` | 数学表达式求值 | `{ expression: string }` |
| `datetime` | 获取当前日期时间 | `{ format?: string }` |
| `fetch_url` | HTTP GET 获取网页文本 | `{ url: string, max_length?: number }` |

`ToolRegistry`：`Vec<Box<dyn Tool>>`，`list_tools() → Vec<FunctionDef>`，`execute(name, args) → String`

### 4. Server 层 (`src/server/`)
| 端点 | 方法 | 请求体 | 响应 |
|------|------|--------|------|
| `/v1/chat` | POST | `{ messages, model?, max_turns? }` | `{ reply: string }` |
| `/v1/chat/stream` | POST | 同上 | SSE: `data: {token}` / `data: [DONE]` |
| `/v1/health` | GET | - | `{ status: "ok" }` |

### 5. CLI 层 (`src/cli/`)
- rustyline REPL，提示符 `> `
- 流式逐 token 绿色打印
- 工具调用过程黄色提示：`[Tool] calculator → 42`
- 斜杠命令：`/quit`、`/clear`、`/tools`

## 环境变量
| 变量 | 默认值 | 说明 |
|------|--------|------|

## 实施步骤
1. `cargo init`，写 Cargo.toml 依赖
2. `src/llm/message.rs` — 消息类型
3. `src/llm/client.rs` + `mod.rs` — OpenAI 客户端（chat + stream）
4. `src/tools/` — Tool trait + ToolRegistry + 3 个示例工具
5. `src/agent/` — AgentConfig + Agent ReAct 引擎
6. `src/cli/mod.rs` — 交互式终端
7. `src/server/` — axum 路由 + SSE 流式
8. `src/main.rs` + `src/lib.rs` — 入口整合
9. `README.md` — 使用说明
10. `cargo build` 编译验证
