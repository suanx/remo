# Awaken 框架 7 大功能拓展执行计划

## Context

Awaken 是一个成熟的 Rust AI Agent 框架（0.5.1-dev），采用 Phase-based 执行引擎 + Plugin 架构。现有 19 个 crate，覆盖 agent 运行时、协议（A2A/MCP/AG-UI）、工具权限、可观测性等。用户要求一次性实现全部 P0-P2 拓展：记忆系统、RAG 管线、多模态、Workflow 引擎、安全沙箱、Playground 工具、Web UI 组件库。

## 架构关键模式（所有新 crate 必须遵循）

```rust
// 1. Plugin trait (crates/awaken-runtime/src/plugins/lifecycle.rs:81)
impl Plugin for MyPlugin {
    fn descriptor(&self) -> PluginDescriptor { ... }
    fn register(&self, registrar: &mut PluginRegistrar) -> Result<(), StateError> {
        registrar.register_key::<MyKey>(StateKeyOptions { persistent, scope, ... })?;
        registrar.register_phase_hook(ID, Phase::BeforeInference, MyHook)?;
        registrar.register_tool_gate_hook(ID, MyGateHook)?;
        Ok(())
    }
    fn config_schemas(&self) -> Vec<ConfigSchema> { ... }
    fn on_activate(&self, agent_spec: &AgentSpec, patch: &mut MutationBatch) -> Result<(), StateError> { ... }
}

// 2. StateKey pattern (awaken-ext-permission/src/state.rs:67)
pub struct MyKey;
impl StateKey for MyKey {
    const KEY: &'static str = "my_key";
    const MERGE: MergeStrategy = MergeStrategy::Commutative;
    const SCOPE: KeyScope = KeyScope::Thread;
    type Value = MyValue;
    type Update = MyAction;
    fn apply(value: &mut Self::Value, update: Self::Update) { value.reduce(update); }
}

// 3. PhaseHook (crates/awaken-runtime/src/hooks/phase_hook.rs:11)
#[async_trait]
pub trait PhaseHook: Send + Sync + 'static {
    async fn run(&self, ctx: &PhaseContext) -> Result<StateCommand, StateError>;
}
```
